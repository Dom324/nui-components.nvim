## Prompt

`Prompt` is a custom single-line [`TextInput`](/docs/components/text-input) component that includes a `prefix` and an `on_submit` callback.

![](/gifs/prompt-1.gif)

### Usage Example

```lua
n.prompt({
  prefix = " > ",
  border_label = {
    text = "Command",
    align = "center",
  },
  on_submit = function(value)
    print(value)
  end,
})
```
