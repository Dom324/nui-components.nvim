## Rows

`Rows` is a component built on top of [`Box`](/docs/components/box), it allows you to arrange children vertically.

```
    ┌───────────────────────────────────────────────────────────┐
 │  │                                                           │
 │  │                                                           │
 │  │                                                           │
 │  └───────────────────────────────────────────────────────────┘
 │  ┌───────────────────────────────────────────────────────────┐
 │  │                                                           │
 │  │                                                           │
 │  │                                                           │
 │  └───────────────────────────────────────────────────────────┘
 │  ┌───────────────────────────────────────────────────────────┐
 │  │                                                           │
 │  │                                                           │
 ▼  │                                                           │
    └───────────────────────────────────────────────────────────┘
```

```lua
n.rows(<children>) -- flex = 1
n.rows(2, <children>) -- flex = 2
n.rows({ flex = 3 }, <children>)
n.rows({ flex = 0 }, <children>)
n.rows({ size = 10 }, <children>)
```
